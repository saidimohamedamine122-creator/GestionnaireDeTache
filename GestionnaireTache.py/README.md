# Gestionnaire de tâches en Python

Ce gestionnaire de tâches permet d'ajouter, supprimer des tâches, de définir leur niveau de priorité et de trier les tâches en conséquence. 
Il permet également de marquer les tâches comme terminées et sauvegarde automatiquement les données.

## Fonctionnalités

- Ajouter une tâche
- Supprimer une tâche
- Marquer une tâche comme terminée
- Trier les tâches selon leur niveau de priorité (Haute / Moyenne / Basse)
- Sauvegarde automatique dans un fichier JSON

## Installation et utilisation

1. Télécharger : `main.py` `Tache.py` `Storage.py`
2. Installer Python si nécessaire : [https://www.python.org/downloads/](https://www.python.org/downloads/)
3. Lancer le programme depuis le terminal ou l’invite de commandes :

```bash
python main.py
